Detailed Description

Exponentially Decreasing Harmonics Oscillator Synth

Ruby version: Synths::Edho

This produces a wave corresponding to the equation

 *
 * inf  n
 *  E  b  * sin((n + 1)wt)
 * n=0
 *
 *

Where b is "brightness", valued from 0 (a sine wave, sine.h) to 1 (a cotangent/2 wave, containing equal amounts of all frequencies, cot.h).

If the scale parameter is set, the amplitude of the wave will be decreased such that the peak is always 1.0.
