Detailed Description

Parabolic Wave Synth

Ruby version: Synths::Parabola

Makes a wave of the equation:

 2                         2
π ( (ft - floor(ft)) - 0.5)  - 1/12 )

Which is equivalent to:

inf             2
 Σ sin(n*wt) / n
n=1
